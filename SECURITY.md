# Security Policy

## 1. Our Commitment
Security is the core of AEGIS. We take every vulnerability report with the highest level of institutional priority.

## 2. Supported Versions
| Version | Supported |
| ------- | --------- |
| 1.0.x   | ✅ Yes    |
| < 1.0   | ❌ No     |

## 3. Reporting a Vulnerability
**Do not open a public GitHub issue for security vulnerabilities.**

Please report any sensitive findings directly to the IA_AUT_ECO Security Team at:
**security@ia-aut-eco.fr**

* **Encryption:** We strongly recommend using our PGP key for communication [Link to PGP Key].
* **Response Time:** You will receive an initial acknowledgment within 24 hours.
* **Disclosure:** We follow a coordinated disclosure policy. We will work with you to patch the flaw before public announcement.

## 4. Scope
This policy covers the AEGIS Framework logic and its integration with Winterhold OS. Hardware-specific vulnerabilities should be reported to the respective manufacturers (NVIDIA, ARM, etc.).
